Compilador con gramatica LR

Compilador / Lenguaje Personalizado → Generación de Código en C

Aplicación tipo compilador desarrollado en Java que permite escribir programas usando un lenguaje personalizado, definido mediante una gramática formal. El sistema valida el código por etapas (léxico, sintáctico y semántico) y genera como salida un archivo con código equivalente en C, respetando la estructura y reglas del lenguaje.

Funcionalidades principales

  • Declaración y asignación de variables: permite declarar variables al inicio del programa con tipos int, float y char. Se admite declaración simple o con asignación inicial.
  • Operaciones aritméticas: soporte de expresiones con +, -, *, / y uso de paréntesis ().
  • Condicionales (if / else): estructuras if (condición) { ... } y else { ... } con operadores <, >, <=, >=, ==, !=.
  • Bucles (for): estructura for (inicialización; condición; actualización) { ... }, incluyendo incrementos y decrementos como id++, id--, ++id, --id.
  • Estructuras anidadas: soporte de anidación de condicionales y bucles (por ejemplo, if dentro de if, for dentro de for, etc.).

Proceso general

  • Análisis léxico: identificación de tokens (identificadores, números, tipos, operadores y símbolos).
  • Análisis sintáctico: validación de la estructura del programa según la gramática definida.
  • Validación semántica: verificación de coherencia (uso correcto de variables y tipos).
  • Generación de código: traducción del programa a un archivo final en C.

Resultado

Herramienta capaz de recibir código en un lenguaje propio, validarlo y generar como salida un programa equivalente en C, soportando variables, operaciones, condicionales y bucles.

CatCode

CatCode es un entorno de desarrollo creado en Java diseñado para crear, analizar y ejecutar programas escritos en un lenguaje personalizado. Permite verificar si el código cumple con la sintaxis establecida, mostrando de forma clara el comportamiento del programa mediante una interfaz organizada e intuitiva.

El sistema realiza validaciones por etapas, incluyendo análisis léxico, sintáctico y semántico, lo que ayuda a detectar errores y comprender la estructura del código. Además, incorpora una ejecución guiada que permite visualizar información clave durante el proceso, como el archivo activo, la salida en consola, los componentes identificados y la pila de ejecución.

Características principales

  • Creación y edición de archivos .mal: permite trabajar con programas en el formato del lenguaje, organizando el código de manera práctica.
  • Análisis léxico, sintáctico y semántico: verificación completa del programa para validar estructura, tokens y coherencia.
  • Ejecución y visualización paso a paso: seguimiento del comportamiento del código para comprender su funcionamiento.
  • Interfaz por pestañas: navegación cómoda entre Consola, Componentes, Pila y el archivo activo.

Ejemplo de código

program hola; 
write("Hola mundo"); 
endp

CatCode facilita el aprendizaje y la comprensión de cómo se analiza y ejecuta un lenguaje, integrando validación, ejecución y visualización dentro de una misma herramienta.

Bilioteca COBAEM

El Sistema de Biblioteca es una aplicación de escritorio desarrollada en Visual Basic .NET, diseñada para gestionar de manera eficiente el registro, préstamo y devolución de libros. Está conectada a una base de datos SQL Server, lo que permite almacenar la información de libros, usuarios y préstamos de forma segura y estructurada.

Características principales

  • Gestión de libros: agregar, editar, eliminar y consultar información.
  • Gestión de usuarios: registrar lectores o socios de la biblioteca.
  • Control de préstamos: registrar préstamos, devoluciones y fechas límite.
  • Interfaz amigable: desarrollada con Windows Forms y Visual Basic .NET.

Tecnologías utilizadas

  • Lenguaje: Visual Basic .NET
  • Entorno de desarrollo: Visual Studio
  • Base de datos: SQL Server
  • Conexión: SqlClient
  • Arquitectura: Windows Forms Application

Monitor de producción ECOPAC

El Monitor de Producción es un sistema desarrollado en Python para ECOPAC, el cual muestra en tiempo real el estado de las máquinas, la producción por turno y los kilos fabricados. Permite supervisar el rendimiento, detectar fallas, optimizar procesos y mantener un control visual y actualizado de la eficiencia en planta.

Funciones principales

  • Visualización en tiempo real del estado de las máquinas.
  • Monitoreo de producción por turno.
  • Registro y control de kilos fabricados.
  • Detección de fallas para una respuesta rápida.
  • Apoyo en la optimización de procesos y rendimiento.

Tecnologías utilizadas

  • Lenguaje: Python
  • Proyecto: ECOPAC

Proyecto Nombre

ColibriView es una aplicación móvil diseñada para la detección del daltonismo, que utiliza pruebas visuales y la inteligencia artificial para ofrecer diagnósticos precisos. La app se enfoca en proporcionar una experiencia de usuario inclusiva e intuitiva, facilitando el proceso de identificación de diferentes tipos de daltonismo.

Creación del proyecto

  • Aplicación móvil: Desarrollada en Android Studio.
  • Página web: Creada como sitio de documentación para explicar el uso, funciones y objetivo del proyecto.

Características de la aplicación

  • Pruebas Visuales: Incluye pruebas de Ishihara, Protan y Deutan, Tritan y Lantern.
  • Inteligencia Artificial: Utiliza algoritmos avanzados para mejorar la precisión del diagnóstico.
  • Interfaz Intuitiva: Diseño enfocado en la accesibilidad y facilidad de uso.
  • Información Educativa: Proporciona información detallada sobre el daltonismo y cómo manejarlo.

Página web de documentación

El proyecto cuenta con una página web que funciona como documentación, donde se presenta información sobre el daltonismo, el propósito de la aplicación, su funcionamiento, las pruebas incluidas y una guía para que los usuarios comprendan cómo utilizarla correctamente.

Beneficios

  • Para Usuarios: Ayuda a entender mejor su condición y a recibir un diagnóstico temprano.
  • Para Médicos: Apoya en el diagnóstico y seguimiento de pacientes con daltonismo.
  • Para Educadores: Facilita la identificación de estudiantes con daltonismo y la adaptación de métodos de enseñanza.

Tecnologías utilizadas

  • Aplicación: Android Studio
  • Lenguaje: Kotlin
  • Web: HTML, CSS, JavaScript, Bootstrap
